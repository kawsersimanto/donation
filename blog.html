Important FAQ

What is Document Object Model (DOM)?

The DOM is a way for programming languages like JavaScript to interact with HTML, letting you change the content and structure of a web page.

How do you select an element from the DOM?

You can select elements using methods like getElementById for IDs, getElementsByClassName for classes, or querySelector for more complex selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation lets you add one event listener to a parent element instead of multiple listeners to each child, which makes your code cleaner and faster.

How do you manipulate an element's attributes and styles using DOM?

You can change attributes using setAttribute and styles by accessing the style property, like element.style.color = 'blue' to make text blue.
